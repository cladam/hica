About the hica Playground

This playground compiles hica source code to JavaScript entirely in your browser. No server is involved. The hica compiler itself is compiled to JS via Koka's --target=js and bundled into a single file.

What works supported

  • Functions, closures, recursion, lambdas
  • If/else, match expressions, guards
  • For-loops, while-loops
  • Structs & enums (algebraic types)
  • Lists, tuples, slice patterns
  • Pipe operator (|>), string interpolation
  • Maybe (Some/None) and Result (Ok/Err)
  • Standard math, string, and list functions

Known limitations not supported

  • max(), min() with multiple arguments are limited
  • Map functions (map_get, map_set, map_keys, etc.) are not available
  • File I/O (read_file, write_file)
  • CLI args (get_args) – no command line in the browser
  • Interactive input (readline)
  • Multi-module imports
  • Algebraic effects / handlers
  • External Koka/C interop (extern)

Tips

  • Press Ctrl+Enter (or Cmd+Enter) to run
  • Use the Examples dropdown to try working programs
  • Click Share to copy a URL with your code embedded
  • See the Language Reference for full syntax documentation
